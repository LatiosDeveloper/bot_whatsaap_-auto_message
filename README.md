# bot_whatsaap_-auto_message
# ☕ Café La Humancia - Automatización WhatsApp

Sistema automático de notificación de pedidos terminados mediante WhatsApp.

## ✨ Funcionalidades

- Notifica automáticamente cuando un pedido está listo (`terminado = si`)
- Control por **ID de pedido** (soporta clientes recurrentes)
- No repite notificaciones
- Mensajes personalizados con emojis
- Compatible con **Brave Browser**

## Requisitos

- Python 3.10 o superior
- WhatsApp Web abierto en Brave
- Archivo `pedidos_unificados_cafe.csv`
